General Routing

Path HTTP Methods Description
/ GET retrieve data from index.html

HTTP Methods for Users

Path HTTP Methods Description
/users GET Retrieves a list of users.
/register POST Creates a new user.
/users/:id GET Retrieves details of a specific user.
/users/:id PATCH Updates details of a specific user.
/users/:id DELETE Deletes a specific user.

HTTP Methods for Products

Path HTTP Methods Description
/shop GET Retrieves a list of shop.
/update POST Creates a new product.
/shop/:id GET Retrieves details of a specific product.
/shop/:id PATCH Updates details of a specific product.
/shop/:id DELETE Deletes a specific product.
